## 安装使用

### 1. 安装须知

- 请参考上面第一条链接
- 在硬件上，还需要 CPU 支持（ Intel 虚拟化技术 (Intel VT) 或 AMD 虚拟化 技术(AMD-V))
- Windows Sandbox 的开启需要一段时间，就像虚拟机启动一样。
- 进入 Windows Sandbox，你可以像日常操作那样，就像在实体系统中那样，在 Windows Sandbox 进行软件的安装、浏览网页等等的操作，一切就跟在实体机操作一模一样。
