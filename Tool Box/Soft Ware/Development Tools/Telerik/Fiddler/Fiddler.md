```c#
1. 打开Fiddler，点击Tools-->Options
2. 点击HTTPS，勾选Decrypt HTTPS traffic和Ignore server certificate(unsafe)
3. 点击Actions，点击Export Root Certificate to Desktop
4. 导出的证书，拖进模拟器中
5. 网络桥接模式开启,重启动模拟器.

```
