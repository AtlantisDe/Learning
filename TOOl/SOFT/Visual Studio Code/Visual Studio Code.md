## Visual Studio [插件配置快捷键]

```shell
全部折叠 "ctrl+m ctrl+m"
递归方式展开 "ctrl+m ctrl+o"
Sync:上传设置 "shift+alt+u"
Past Url "ctrl+alt+p"
删除行注释 "ctrl+k ctrl+u"
复制文件路径打开文件 "ctrl+o"
```
