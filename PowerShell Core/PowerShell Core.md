# PowerShell Core

```c#
多年来，PowerShell 已有很多版本发布。 最初，Windows PowerShell 是在 .NET Framework 基础之上构建而成，仅适用于 Windows 系统。 在最新版本中，PowerShell Core 使用 .NET Core 2.x 作为运行时。 PowerShell Core 支持 Windows、macOS 和 Linux 平台。
```
