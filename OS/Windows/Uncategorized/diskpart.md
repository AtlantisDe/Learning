# diskpart

- [联想中国(Lenovo China)联想知识库](http://iknow.lenovo.com/detail/dc_148698.html)

## 常用

```c#
// 输入“list disk”回车，查看当前的磁盘状况；
list disk

// 输入sel disk 0回车，选中该磁盘；
sel disk 0

// 再输入 list par 查看分区界面；
list par

// 然后输入sel par x x代表选择的分区；
sel par 1


// 再输入 del par 删除分区。
del par

```
