# js switch

## 常用

```c#
switch (guid) {
  case "Unknown":
    vue = '<font color="#FF0000">未知</font>';
    break;
  case "Started":
    vue = '<font color="#8000ff">已启动</font>';
    break;
  case "Starting":
    vue = '<font color="#FF0000">启动中</font>';
    break;
  case "Stopped":
    vue = '<font color="#FF0000">已停止</font>';
    break;
  case "Stopping":
    vue = '<font color="#FF0000">停止中</font>';
    break;
  case "nosite":
    vue = '<font color="#FF0000">无该站点</font>';
    break;
}
```

## demo

```c#
switch (guid) {
  case "Unknown":
    break;
  case "Started":
    break;
  default:
    break;
}
```
