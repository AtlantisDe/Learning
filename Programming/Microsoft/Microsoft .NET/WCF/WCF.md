# WCF

- Windows Communication Foundation(WCF)是由微软开发的一系列支持数据通信的应用程序框架，可以翻译为 Windows 通讯开发平台。
- 整合了原有的 windows 通讯的 .net Remoting，WebService，Socket 的机制，并融合有 HTTP 和 FTP 的相关技术。
- 是 Windows 平台上开发分布式应用最佳的实践方式。
