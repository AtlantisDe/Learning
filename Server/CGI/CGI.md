# CGI

- CGI 的全称是 Common Gateway Interface（通过网关接口）。它是一个接口标准，通过这个标准，Web 服务器可以将请求转发给第三方程序，由第三方程序生成 Web 页面，然后 Web 服务器将这个生成的页面返回给客户端。
