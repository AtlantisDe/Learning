## 常用

```c#
// Console使用Serilog
Install-Package Serilog
Install-Package Serilog.Sinks.Console

Install-Package Serilog
Install-Package Serilog.Sinks.Console
Install-Package Serilog.Sinks.File

// 在Web程序中使用Serilog
Install-Package Serilog
Install-Package Serilog.AspNetCore
Install-Package Serilog.Sinks.Console
Install-Package Serilog.Sinks.File
Install-Package Serilog.Settings.Configuration

```
