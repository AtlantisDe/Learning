# Nancy

- Nancy 官网并没有明确的说 Nancy 是一个 MVC 框架，
- 官方的定义是：Nancy 是一个用于构建基于 HTTP 的 Web 服务的框架。我题目中所写的轻量级 MVC 框架有些不妥之处，但是可以让大家快速的对 Nancy 有个大致的了解。不过，说实话，Nancy 和 asp.net 在一起，实在是太像 MVC 了。
