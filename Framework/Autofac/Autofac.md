# Autofac

- Autofac 是一款适用于 Microsoft .NET 4.5, Silverlight 5, Windows Store apps, and Windows Phone 8 apps 的超赞的 IoC 容器 . 它可以管理类之间的依赖关系从而使 应用在规模及复杂性增长的情况下依然可以轻易地修改 . 通过将常规.net 类当做 组件 处理实现 .
