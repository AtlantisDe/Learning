# layer.open

-[layer 弹层组件开发文档](https://www.layui.com/doc/modules/layer.html)

## 基础参数

-

```js
layer.open({
      type: 2,
      title: '很多时候，我们想最大化看，比如像这个页面。',
      closeBtn: 1,//0 不显示关闭按钮 1 显示关闭按钮
      shadeClose: true,
      shade: false,
      maxmin: true, //开启最大化最小化按钮
      area: ['893px', '600px'],
      content: '//fly.layui.com/'
    });
  }
```
