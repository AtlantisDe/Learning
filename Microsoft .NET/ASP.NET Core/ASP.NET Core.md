# ASP.NET Core

- .NET Core 是微软开发出的另一个可以跨 Linux、Windows、Mac OS 等平台的.NET。
- ASP.NET Core 是一个跨平台的高性能开源框架，用于生成基于云且连接 Internet 的新式应用程序。
